About Me

My main focus is on developing innovative Web-based software using principled design and construction techniques. Currently I am producing demos and experimenting with the following:

  • Sharing client and server side code written in JavaScript (using Rhino or V8 (Node.js) as the server side interpreter).
  • Amazon EC2 and S3.
  • Vertx, an asynchronous application framework including WebSockets.
  • Handlebars for client and server side HTML templating.
  • CSS styling in LESS.
  • JavaScript frameworks including WebGL and in particular three.js.

I am also developing an interest in the open source communities revolving around these areas.
